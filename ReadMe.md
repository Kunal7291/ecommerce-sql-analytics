# E-Commerce SQL Analytics Project

## Objective
Analyze real-world e-commerce data using SQL to generate business insights.

## Dataset
Brazilian E-Commerce Dataset (Olist)

## Tools Used
- MySQL
- SQL (Advanced queries, window functions)

## KPIs
- Total Revenue
- Total Orders
- Average Order Value

## Advanced SQL
- LAG (growth analysis)
- RANK / DENSE_RANK
- CASE statements

## Insights
- Identified high-value customers
- Analyzed category performance
- Measured customer retention
